# Historial-del-usuario
Para hacer un instructivo de uso de inventario.py, necesito ver el archivo o saber cómo funciona. Un instructivo depende de cosas como:

Qué hace el programa (gestión de productos, stock, ventas, etc.)

Cómo se ejecuta (por ejemplo: python inventario.py)

Qué opciones o menús tiene

Qué datos pide al usuario

Qué archivos genera o modifica

 Envíame una de estas opciones:

El archivo inventario.py completo, o

Copia y pega el código aquí, o

Una captura del programa funcionando o del menú.

Con eso puedo prepararte un instructivo claro, por ejemplo:

Introducción al sistema

Requisitos

Cómo ejecutar el programa

Menú de opciones

Ejemplos de uso

Solución de errores comunes
